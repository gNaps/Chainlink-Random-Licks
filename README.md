# Random Licks

> Generate random licks for musical inspiration

---


This dapp allows you to generate a Random Licks using Chainlink VRF.

## Get Started

Download
```
git clone https://github.com/gNaps/Chainlink-Random-Licks
```

### Install all deps
```
yarn
```

### Generate subgraph schema
```
yarn codegen 
```

### Run frontend
```
yarn dev run
```
